# Copy Variants - Portfolio Senior Full Stack

## Análisis del Copy Actual

**Actual implementado:**
- Badge: "Senior Fullstack Engineer"
- Headline: "Miguel Barra"
- Subheadline: "Construyo software que escala y perdura."
- Credencial: "+5 años transformando ideas en productos digitales reales."

**Problema:** El nombre como headline no comunica valor diferencial en los primeros 3 segundos.

---

## Propuestas de Copy

### VARIANTE A: Orientada a Resultados (Recomendada)

```
Badge:       SENIOR FULLSTACK ENGINEER
Headline:    Del problema al producto.
Subheadline: Arquitectura que escala. Código que perdura.
Credencial:  +5 años llevando ideas de startups a producción.
```

**CTAs:**
- Primario: "Ver proyectos"
- Secundario: "Descargar CV"

**Justificación:**
- **"Del problema al producto"** → Comunica el journey completo, no solo código
- **Orientado a founders** → Entienden "problema → producto" como su necesidad
- **Recruiters técnicos** → "Arquitectura" y "escala" son keywords de seniority
- **Doble beneficio** → Escala (negocio) + Perdura (técnico)

---

### VARIANTE B: Orientada a Diferenciación Técnica

```
Badge:       FULLSTACK ENGINEER · CHILE
Headline:    Ingeniería primero. Código después.
Subheadline: Diseño soluciones que evitan deuda técnica desde el día uno.
Credencial:  Ingeniero en Computación · +5 años en producción.
```

**CTAs:**
- Primario: "Explorar stack"
- Secundario: "Contactar"

**Justificación:**
- **"Ingeniería primero"** → Diferencia de "coders", posiciona como ingeniero
- **Anti-deuda técnica** → Pain point real de CTOs y tech leads
- **Badge con ubicación** → Útil para búsquedas regionales (LATAM)
- **Tono más técnico** → Resuena con equipos de ingeniería

---

### VARIANTE C: Orientada a Impacto de Negocio

```
Badge:       SENIOR FULLSTACK · REMOTE-READY
Headline:    Software que funciona. Equipos que avanzan.
Subheadline: Desarrollo end-to-end para startups que no pueden perder tiempo.
Credencial:  De idea a MVP en semanas, no meses.
```

**CTAs:**
- Primario: "Agendar llamada"
- Secundario: "Ver CV"

**Justificación:**
- **"Funciona + Avanzan"** → Doble promesa: técnica y humana
- **"Startups que no pueden perder tiempo"** → Targeting directo, urgencia implícita
- **"Semanas, no meses"** → Promesa concreta, diferenciadora
- **CTA "Agendar llamada"** → Conversión directa para founders
